File Shredder

File Shredder – бесплатный файловый шреддер, предназначенный для безвозвратного удаления информации с различных носителей. Примечание: программа считается устаревшей и не рекомендуется для использования на современных SSD-дисках.

Функционал по удалению данных:

  • Стирание как файлов, папкок, так и всего носителя.
  • Удаление лишней информации со свободного места на носителе.
  • Выбор метода стирания данных с носителей.

Существует в одной версии:

  • Free

Способы удаления данных:

  • Simple (1 проход)
  • Simple (2 прохода)
  • DoD 5220-22.M (3 переходов)
  • Secure erasing (7 проходов)
  • Gutmann (35 проходов)
